# Best Practices

The quality of generated speech depends heavily on how your text is written. Well-structured text with proper punctuation and correct spelling produces significantly more natural results.

This guide explains how to write text for the best voice quality, along with examples in multiple languages.

---

## General Guidelines

Always follow these recommendations:

- ✅ Use complete sentences.
- ✅ Add commas and full stops.
- ✅ Use the correct writing script for each language.
- ✅ Check spelling before generating speech.
- ✅ Write numbers the way you want them to be spoken.
- ✅ Keep paragraphs well structured.
- ✅ Use proper capitalization where applicable.

Avoid:

- ❌ ALL CAPITAL LETTERS
- ❌ Random punctuation (`!!!!????`)
- ❌ Long text without commas or periods
- ❌ Incorrect spellings
- ❌ Unsupported symbols or corrupted text

## Example 1 — English

### ❌ Poor

```text
hello everyone today i will explain artificial intelligence lets start
```

### ✅ Better

```text
Hello everyone! Today, I will explain Artificial Intelligence. Let's get started.
```

# Use the Native Script

For the best pronunciation, always write text in the language's native writing system.

| Language | Recommended |
|----------|-------------|
| English | Latin |
| Hindi | Devanagari |
| Telugu | Telugu |
| Tamil | Tamil |
| Bengali | Bengali |
| Urdu | Arabic Script |
| Arabic | Arabic |
| Chinese | Chinese Characters |
| Japanese | Japanese |
| Korean | Hangul |
| Russian | Cyrillic |

# Writing Numbers

Numbers can sometimes be pronounced differently depending on the context.

### Numeric

```text
I have 1250 subscribers.
```

### Written

```text
I have one thousand two hundred and fifty subscribers.
```

If pronunciation is important, consider writing the number in words.

---

# Using Punctuation

Proper punctuation creates more natural pauses.

### ❌ Without punctuation

```text
Welcome everyone today we are going to learn machine learning thank you for joining
```

### ✅ With punctuation

```text
Welcome, everyone! Today, we are going to learn about machine learning. Thank you for joining.
```

---

# Long Paragraphs

Instead of writing one very long paragraph, split your content into smaller sections.

### Recommended

```text
Welcome to our tutorial.

Today, we'll learn the basics of artificial intelligence.

Let's begin with a simple example.
```

This improves rhythm and makes the speech sound more natural.

---

# Mixed Languages

Mixed-language content is supported in many cases.

Example:

```text
Hello everyone! आज हम AI के बारे में सीखेंगे.
```

For the best pronunciation, avoid switching languages too frequently within the same sentence.

---

# Common Mistakes

Avoid these common issues:

- Writing without punctuation
- Incorrect spelling
- Repeating punctuation (`!!!!!!`)
- Using random emojis throughout the text
- Excessive CAPITAL LETTERS
- Extremely long sentences
- Unsupported characters

---

# Tips for Professional Voiceovers

- Write conversationally.
- Keep sentences short and clear.
- Use punctuation to control pauses.
- Break long scripts into sections.
- Preview the generated audio before downloading.
- Make small edits if pronunciation is not exactly as expected.

Following these practices will help you produce smoother, clearer, and more natural-sounding speech across all supported languages.
